Description

Created an engaging and user-friendly to-do list website from scratch. The site was designed with a modern and visually appealing look in Figma. It's built using EJS, CSS, and JavaScript for the front-end, and Node.js with Express.js for the back-end, ensuring smooth and responsive user interactions. This website empowers users to efficiently manage their tasks and enhance their productivity.

Key Features

  • 1.
    Task Creation
    Users can easily create new tasks by simply pressing the 'Enter' key, streamlining the process and making task management effortless. This feature ensures that adding tasks is a seamless and efficient experience.
  • 2.
    Task Completion
    By clicking on a task, users can instantly cross it out, marking it as completed. This feature enhances task management and keeps users in control of their to-do lists with ease.
  • 3.
    Dynamic Time Display
    The dynamic time display feature keeps users informed and organized, automatically updating and displaying the current date and time. Users can easily track the date as they manage their tasks, ensuring they stay on top of their schedules effortlessly.
